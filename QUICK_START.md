# 🚀 VINCI-ARENA PRO - Quick Start Guide

## Start Development
```bash
# Terminal 1: Backend
cd ~/Projects/tournament-calculator/backend
npm run dev

# Terminal 2: Frontend  
cd ~/Projects/tournament-calculator/frontend
npm run dev

# Open: http://localhost:3000
```
